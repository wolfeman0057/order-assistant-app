# Order Assistant — iPhone-installable PWA (GitHub Pages build)

Live app: **https://wolfeman0057.github.io/order-assistant-app/**

Open that URL in Safari on iPhone → Share → **Add to Home Screen** → launch "Orders".

This repo IS the deployment: GitHub Pages serves the static PWA directly from `main`.
The AI backend (order lookup tools + GPT-4o, key server-side) runs at
https://order-assistant-et60.onrender.com (source: github.com/wolfeman0057/order-assistant).
